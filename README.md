#  Sidebar Navigation UI (Pure CSS)

An elegant and lightweight sidebar navigation interface built using **HTML and CSS only** — no JavaScript required. This project showcases how far modern CSS can go in creating interactive UI components using simple techniques like the checkbox toggle pattern.

---

##  Overview

This project demonstrates a fully functional **toggleable sidebar menu** using pure CSS. It mimics real-world UI patterns found in modern web applications while keeping the implementation minimal and efficient.

---

##  Key Highlights

-  Pure CSS interactivity (no JavaScript)
-  Clean and modern UI design
-  Lightweight and fast
-  Responsive-friendly layout
-  Smart use of CSS selectors (`:checked`)

---

##  Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & interactivity  
- **Font Awesome** – Icons  
- **Google Fonts** – Typography  

---

##  Core Concept

The sidebar toggle is controlled using:

```css
#check:checked ~ .sidebar_menu {
    left: 0;
}
```
